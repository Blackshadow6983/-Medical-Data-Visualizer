# Medical Data Visualizer

A Python/pandas/seaborn project that visualizes cardiovascular examination data — comparing lifestyle and health indicators against cardiovascular disease presence, and exploring correlations between features.

## Files
- `medical_data_visualizer.py` — main analysis and plotting functions
- `main.py` — runs the visualizer and tests
- `test_module.py` — unit tests
- `medical_examination.csv` — dataset
- `catplot.png`, `heatmap.png` — example outputs

## Run it
```bash
python3 main.py
```

Built as part of [freeCodeCamp's Data Analysis with Python](https://www.freecodecamp.org/learn/data-analysis-with-python/) certification.
